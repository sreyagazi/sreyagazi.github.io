Character Design Iterations, Expressions & Turnaround


↑ Some design / costume iterations for my original character before I ultimately chose the design that I did. I created this character last year, during the summer of 2024, for the purpose of character animation practice so that I can improve my skills in emotive storytelling, as I made her with the intention for her to be especially expressive.

Character Design & Clay Modelling


↑ This is some of my earliest character design work. I painted the above painting on the left using acrylic paint when the idea for the character first came to me. The turnarounds were drawn in a sketchbook and coloured with copic markers.
This was long before I ever touched Blender or other 3D software programs, so I modelled the characters using clay and with toothpicks for bones as I really wanted to visualise them both in 3D.
